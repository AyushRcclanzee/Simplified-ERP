# Getting Started

## About

An Ecommerce website.

## Tech Stack

1. Frameworks
    -  ReactJs
    - Tailwind CSS
2. Libraries & Dependencies
    - Framer Motion
    - Lucide React
3. Hosting & Database
    - Vercel
    - GitHub

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install.

clone the repo :
```bash
git clone 
```

Install dependencies :
```bash
npm i
```

## Usage

```bash
npm start
```
